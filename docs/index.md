## Current Principles

- Official/public sources are the only acceptable RAG evidence.
- Raw user questions and personal data are not stored.
- Provider, deployment, and database decisions remain gated until documented evidence closes them.
- Documentation changes that affect behavior, safety, source policy, or deployment assumptions must land with the related code change.
