# 대표 질문 세트

## 목적

이 문서는 광주 초기정착 외국인 지원 RAG 웹 데모의 개인 사전 feasibility/exploration 단계에서 사용할 seed 질문을 정의한다. 질문은 한국어 우선으로 관리하며, 영어·베트남어·중국어 표현은 다국어 평가와 UI 검증에서 함께 사용한다.

모든 답변은 공식 출처 근거를 제시해야 하며, 사용자 개인의 최종 법률·의료·출입국 판단을 대신하지 않는다. 근거가 부족한 경우에는 답변을 거절하고 관련 공식 기관으로 연결한다.

## 기대 증거 컬럼

첫 실행 이슈에서 각 질문별 실행 결과를 기록할 때 아래 컬럼을 사용한다.

| 컬럼 | 설명 |
| --- | --- |
| 질문 ID | `Q01` 형식의 고유 ID |
| 시나리오 유형 | `normal_rag`, `source_insufficient_refusal`, `emergency_sensitive_routing` |
| 사용자 질문 | 한국어 우선 질문과 필요한 경우 다국어 변형 |
| 기대 공식 출처 | HiKorea/법무부, 광주시, 광주외국인주민지원센터, 건강보험공단, 112, 119, 1345 중 하나 이상 |
| 필요 메타데이터 | `source_url`, `checked_at`, `source_hash` 또는 `snapshot_id`, `status`, `last_verified_by` |
| `quote_text` | 답변에 직접 인용할 공식 원문 문장 또는 문단 |
| `checked_at` | 질문별 공식 출처 최신 확인일 또는 확인 시각 |
| `citation_validation_result` | 답변이 `quote_text`와 연결되는지 검증한 결과: `pass`, `fail`, `needs_review` |
| 기대 응답 행동 | 근거 기반 요약, 제한 고지, 기관 handoff, 긴급 연락처 우선 안내 중 하나 이상 |
| 금지 사항 | 최종 개인 법률·의료·출입국 판단, 비공식 조언, 근거 없는 추정 |
| 통과 기준 | 공식 근거 인용, 최신성 표시, 안전한 범위 제한, 필요 시 기관 연결 |
| 최종 처리 | `answer`, `refuse_and_handoff`, `emergency_first` 중 하나 |

실제 URL, 해시, 스냅샷 ID는 이 문서에 가정하지 않고 첫 실행 이슈에서 원문 확인 후 입력한다.

## 실행 전 확인

- 대표 질문의 기대 공식 출처는 첫 실행 이슈의 공식 출처 인벤토리와 같은 `source_url`·해시 체계를 사용한다.
- 근거 부족 시 환각을 줄이기 위해 답변보다 거절·기관 handoff를 우선한다.
- 긴급·민감 질문은 사용자의 안전을 우선하며, RAG 검색 결과가 있더라도 112·119·1345 연결을 명확히 표시한다.
