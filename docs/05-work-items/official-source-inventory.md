# 공식 출처 인벤토리

## 목적

이 문서는 광주 초기정착 외국인 지원 RAG 웹 데모의 첫 실행 전에 반드시 닫아야 하는 공식 출처 게이트를 정의한다. MVP는 개인 사전 feasibility/exploration 단계이며, 실제 수집 URL과 해시 값은 첫 실행 이슈에서 확정한다.

## 공식 출처 원칙

- 한국어 공식 원문을 기준 출처로 삼고, 영어·베트남어·중국어 응답은 공식 번역 또는 검토된 용어집을 통해 보조한다.
- 블로그, 커뮤니티, 비공식 요약, 개인 경험담은 근거로 사용하지 않는다.
- 사용자별 법률·의료·출입국 최종 판단은 제공하지 않고, 해당 기관 안내 및 문의 경로로 연결한다.
- 긴급 상황은 RAG 답변보다 112, 119, 1345 등 즉시 연결 가능한 공식 연락처 안내를 우선한다.

## 필수 메타데이터

모든 수집 단위는 첫 실행 이슈에서 아래 필드를 채워야 한다.

| 필드 | 설명 | 필수 여부 |
| --- | --- | --- |
| `source_url` | 공식 문서, 안내 페이지, 보도자료, 기관 페이지의 실제 URL | 필수 |
| `checked_at` | 사람이 원문을 확인한 ISO 8601 시각 또는 날짜 | 필수 |
| `source_hash` 또는 `snapshot_id` | 원문 스냅샷의 해시 또는 보관 식별자 | 필수 |
| `status` | `active`, `stale`, `retired` 중 하나 | 필수 |
| `last_verified_by` | 마지막 확인자 식별자. 개인 사전 단계에서는 작성자 또는 실행 이슈 담당자 | 필수 |
| `source_name` | 공식 출처명. 예: HiKorea, 광주광역시, 국민건강보험공단 | 필수 |
| `jurisdiction` | 적용 지역·관할. MVP 기본값은 `Gwangju` | 필수 |
| `language` | 원문 또는 공식 번역 언어. 예: `ko`, `en`, `vi`, `zh` | 필수 |
| `topic` | 체류, 건강보험, 지역 생활, 긴급, 상담기관 등 주제 | 필수 |
| `document_type` | 안내 페이지, PDF, 공지, 기관 카드, FAQ 등 문서 유형 | 필수 |
| `reliability_tier` | `official_primary`, `official_secondary`, `agency_referral` 중 하나 | 필수 |
| `update_policy` | 재확인 주기 또는 데모 전 재확인 필요 여부 | 필수 |

실제 `source_url`, `source_hash`/`snapshot_id`, 검증 시각은 이 문서에 임의로 채우지 않는다. 첫 실행 이슈에서 원문을 열람하고 스냅샷을 확보한 뒤 기록한다.

## 청크 메타데이터

RAG 검색에 들어가는 문서 청크는 첫 실행 이슈에서 아래 필드를 함께 채운다.

| 필드 | 설명 | 필수 여부 |
| --- | --- | --- |
| `document_id` | 원문 문서 단위 식별자 | 필수 |
| `chunk_id` | 문서 안의 검색 청크 식별자 | 필수 |
| `quote_text` | 답변에 인용할 수 있는 원문 문장 또는 문단 | 필수 |
| `page_or_section` | 페이지, 제목, 섹션, 앵커 등 위치 정보 | 필수 |
| `embedding_model` | 임베딩 생성에 사용한 모델명 또는 로컬 방식 | 필수 |
| `created_at` | 청크 생성 시각 | 필수 |
| `checked_at` | 원문 최신성 확인 시각 | 필수 |
| `source_hash` 또는 `snapshot_id` | 원문 스냅샷과 청크를 연결하는 해시 또는 식별자 | 필수 |

## 출처 클래스 목록

| 출처 클래스 | 우선 사용 범위 | 첫 실행 확인 항목 | 현재 상태 |
| --- | --- | --- | --- |
| HiKorea / 법무부 | 체류, 외국인등록, 비자, 출입국 민원, 1345 연계 | HiKorea 안내 페이지와 법무부 공식 안내의 최신성, 다국어 제공 여부, 1345 연결 문구 | URL·해시 미확정: 첫 실행 이슈에서 입력 |
| 광주광역시 | 광주 생활정보, 행정서비스, 지역 정책, 민원 연결 | 광주광역시 공식 홈페이지의 외국인·다문화·생활 안내 페이지, 담당 부서, 업데이트 일자 | URL·해시 미확정: 첫 실행 이슈에서 입력 |
| 광주외국인주민지원센터 | 지역 외국인 상담, 통번역, 생활 지원, 프로그램 안내 | 센터 공식 페이지 또는 공식 공지의 상담 범위, 연락처, 운영시간, 언어 지원 | URL·해시 미확정: 첫 실행 이슈에서 입력 |
| 국민건강보험공단 | 건강보험 자격, 보험료, 외국인 가입, 지사·고객센터 안내 | NHIS 공식 외국인 안내, 다국어 자료, 자격·보험료 안내의 최신성 | URL·해시 미확정: 첫 실행 이슈에서 입력 |
| 112 | 범죄, 폭력, 신변 위협, 경찰 긴급 신고 | 경찰청 또는 112 공식 안내의 긴급신고 범위, 외국어 지원 여부 | URL·해시 미확정: 첫 실행 이슈에서 입력 |
| 119 | 화재, 구조, 구급, 응급의료 이송 | 소방청 또는 119 공식 안내의 신고 범위, 응급 상황 안내, 외국어 지원 여부 | URL·해시 미확정: 첫 실행 이슈에서 입력 |
| 1345 | 출입국·외국인 종합안내, 다국어 상담, 기관 handoff | 1345 운영시간, 상담 언어, 연결 범위, HiKorea/법무부와의 일관성 | URL·해시 미확정: 첫 실행 이슈에서 입력 |

## 상태 판정 기준

| 상태 | 의미 | 사용 정책 |
| --- | --- | --- |
| `active` | 공식 원문이 존재하고 최신성·스냅샷이 확인됨 | RAG 근거로 사용 가능 |
| `stale` | 원문은 있으나 최근성, 변경 여부, 해시 확인이 부족함 | 답변 근거로 단독 사용 금지. 기관 handoff 포함 |
| `retired` | 원문이 폐기, 삭제, 대체되었거나 공식성이 사라짐 | 검색 인덱스에서 제외하고 대체 출처 필요 |

## 첫 실행 게이트 체크리스트

- [ ] 각 출처 클래스마다 최소 1개 이상의 공식 `source_url` 확보
- [ ] 각 수집 단위마다 `checked_at` 기록
- [ ] 각 수집 단위마다 `source_hash` 또는 `snapshot_id` 기록
- [ ] 각 수집 단위마다 `status`를 `active`, `stale`, `retired` 중 하나로 판정
- [ ] 각 수집 단위마다 `last_verified_by` 기록
- [ ] 각 수집 단위마다 `source_name`, `jurisdiction`, `language`, `topic`, `document_type`, `reliability_tier`, `update_policy` 기록
- [ ] RAG 청크마다 `document_id`, `chunk_id`, `quote_text`, `page_or_section`, `embedding_model`, `created_at`, `checked_at`, `source_hash` 또는 `snapshot_id` 기록
- [ ] 대표 질문별 답변이 `quote_text`와 연결되는지 citation validation 결과 기록
- [ ] 비공식 출처가 인덱스에 섞이지 않았는지 확인
- [ ] 긴급·민감 질문에서 기관 연결이 우선되도록 대표 질문 세트와 대조
